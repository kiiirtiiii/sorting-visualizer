# sorting-visualizer
> [See the project here](https://kiiirtiiii.github.io/sorting-visualizer/)

Shows the working of different sorting algorithms like:
- Bubble sort
- Selection sort
- Insertion sort
- Merge sort
- Quick sort
- Heap sort
